1 Установить модуль: меню => дополнения => установка дополнений =>GenGraphProduct.ocmod.zip
2 Обновить модификаторы: меню => дополнения => менеджер дополнений 
3 Разрешить редактирование модуля: меню => система => пользователи => группы пользователей => Администратор
4 Для того чтобы вывести модуль: необходимо перейти в дизайн => макеты/схемы => выбрать нужную схему и добавить модуль.
5 для того чтобы вывести в "другом месте" необходимо выполнить пункт 4, затем перейти \catalog\view\theme\default\template\extension\module\gen_graph_product.twig => удалить с шаблона 
<div>
<h3>{{ title_module }}</h3>
  <canvas id="myChart"></canvas>
</div>
=> вставить в том месте, где Вам нужно.
